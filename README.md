xtpl-sublime
============

XTemplate基于sublime2的语法高亮插件

### 使用方法

1. 下载XTpl.tmLanguage文件
2. 把上面文件放到你的sublime/Packages/User目录即可


### roadmap路线图

1. 支持xtemplate的语法高亮所有语法
2. 支持xtempate的语法自动提醒
